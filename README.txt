Требования к серверу бота для подключения и публикации обновлений по URL-адресу, 
который проект предоставляет Telegram, по которому бот прослушивает обновления:
    1.  Поддерживает IPv4. IPv6 пока не поддердивается для веб-перехватчиков
    2.  Принимает входящие POST-запросы из подсетей 149.154.160.0/20 и 
        91.108.4.0/22 через порты 443, 80, 88 или 8443.
    3.  Умеет обрабатывать HTTPS-трафик TLS1.2(+)
    4.  Предоставляет поддерживаемый, проверенный или самозаверяющий сертификат.
    5.  Использует CN или SAN, соответствующий домену, который был указан при 
        настройке при настройке.
    6. Предоставляет все промежуточные сертификаты для завершения цепочки проверки.
